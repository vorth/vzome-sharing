This part is an extension of the Zometool system.&nbsp; It allows you to simulate two G1 struts going into the same hole in a Zome ball, by using this part and two HG2 (half-green) struts instead.&nbsp; This part will not work for other sizes of green struts.<br /> <br /> Note: there are two configurations for a pair of green struts sharing a single hole, a very narrow (acute) angle and a somewhat wider (but still acute) angle.&nbsp; This part models the wider angle.&nbsp; Be sure that this is the one you need.

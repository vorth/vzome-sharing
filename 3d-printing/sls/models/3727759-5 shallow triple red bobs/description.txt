This is a Zometool extension, designed to substitute for a Zome ball where there are three red lines intersecting. &nbsp;Twenty of these triple bobs (as well as 60 double bobs)&nbsp;are necessary for modeling the most symmetric projection of the 6-dimensional hypercube, using all red struts, if one wants to use Zome balls only for the actual projected vertices and not for intersections that result from the projection.<br /> <br /> &nbsp;
